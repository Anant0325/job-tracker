# Job Tracker

Simple Spring Boot + H2 app to track job applications.

## Run locally
```bash
export SPRING_PROFILES_ACTIVE=dev
export JWT_SECRET=dev-secret-change-me
mvn spring-boot:run
```

App runs at http://localhost:8080  
H2 console: http://localhost:8080/h2
